## 使用方法

1. Fork 本仓库到自己的 GitHub 账号
2. 根据需要修改 `configs` 目录下对应的配置文件
3. 如需修改默认 IP、添加插件等，编辑 `diy-script.sh`
4. 进入 **Actions** 页面，选择要编译的 workflow
5. 点击 **Run workflow** 手动触发编译
6. 编译完成后在 [Releases](https://github.com/ilfordarber15/openwrt-ci/releases) 下载固件
